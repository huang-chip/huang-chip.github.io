      # 听说你想在程序退出前执行代码

      # 我的需求

      最近在用micropython玩单片机, 有时候oled屏幕在程序退出后不会熄灭, 只要通电, 他就像一个傻小子一样亮一天。
      因此我想每次退出程序的时候, 关闭oled屏幕。

      # 答案 (atexit内置库)

      • atexit 模块 (opens new window)定义了清理函数的注册和反注册函数. 被注册的函数会在解释器正常终止时执行
      • 使用内置库 atexit 注册程序退出前要执行的函数，即使程序崩溃，都会执行。

      # 三种使用情景

      # 1. 注册函数

      atexit.register(func, *args, **kwargs)

      > 以上为官方提供的注册函数样例

|参数|func|\*args|\**kwargs|
|-|:-:|:-:|:-:|
|**意义**|函数名称|不定数量的参数|不定数量的键值对|

##### 不传参的情况
```python
import atexit
def func_1():
    print('结束')
atexit.register(func_1)# 注册了一个清理函数, 程序结束时运行函数

if __name__ == '__main__':
    print("我来代替程序中正确运行的语句")
    # print(如果你想体验报错语句, 运行我即可)
      # 传参的情况
      import atexit
def func_2(author,lasttime):
    print("本程序作者: ",author, "\t最后编辑于: ",lasttime)
# 传参(*args方式)
atexit.register(func_2,"Mio","2022-10-14")
# OR (**kwargs情况)
atexit.register(func_2,author="Mio二号",lasttime="2022-01-14")

if __name__ == '__main__':
    print("我来代替程序中正确运行的语句")
    # print(如果你想体验报错语句, 运行我即可)

      输出结果

      你可以很直观的看见, atexit输出结束函数时
      符合'栈'的 "先进者后出队, 后来者居上先出队"的规律

      # 2. 装饰器模式

      import atexit

# 只需要在'需要装饰的函数'上方写入即可
@atexit.register
def func_1():
    print('结束')

if __name__ == '__main__':
    print("我来代替程序中正确运行的语句")
	# print(如果你想体验报错语句, 运行我即可)

      # 3.取消注册函数

      atexit.unregister(func)

      如果你不想调用了, 将函数名填入上列'func'中, 即可取消它作为结束函数
      注意 : 无论你之前将某函数注册了几次, 运行这行, 会完全取消这个函数作为结束时的输出, 而不是针对某一次
