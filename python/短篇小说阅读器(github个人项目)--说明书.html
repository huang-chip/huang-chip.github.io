        # 运行代码前, 请安装好以下环境

        1. python编译环境

        可以参照此教程下载Python : Python环境三分钟安装指南 (opens new window)

        1. 相关的第三方库

        按win+R, 在输入框内输入cmd, 按回车后进入到命令行工具中, 依次输入以下语句(一次一行)

        pip install -i https://pypi.tuna.tsinghua.edu.cn/simple requests
pip install -i https://pypi.tuna.tsinghua.edu.cn/simple webbrowser
pip install -i https://pypi.tuna.tsinghua.edu.cn/simple beautifulsoup4
pip install -i https://pypi.tuna.tsinghua.edu.cn/simple pyttsx3

        # 依赖的第三方库介绍

        • requests (爬虫最基础的库) [断网或缺少此库, 则无法get新的短篇小说]
        • webbrowser (用于打开html文件) [缺少此库, 点击"预览"网页无法自带弹出]
        • beautifulsoup4 (一种网页源代码解析库) [缺少此库, 无法实现"阅读"功能]
        • pyttsx3 (文字转机械音) [缺少此库, 无法实现语音合成(阅读)功能]
