            # Python是什么?

            Python是一种灵活多用的计算机程序设计语言, 在2021年, Python被加冕为最受欢迎的编程语言

            • 由于程序员之间无法统一"XX语言是最好的语言"这一问题, 那么就由我个人来谈一谈python的优点
            • Python简单易学，并且免费、开源, 有着简单的语法结构。
            • Python 有很强的可移植性能, 它可以在任何系统, 任何配置了python环境的计算机上执行
            • Python有优秀的开源社区和第三方库, 只要一行简单的pip命令, python的第三方库就能让你眼界大开*(如:Numpy, Pandas, Matplotlib、Requests, Opencv...)*
            • 人生苦短, 我用Python。希望大家都能用上这款优雅、简易的编程语言

            # 三分钟安装python指南

            # 一、从Python官网下载安装包

            • python官网下载链接(Windows) (opens new window) (opens new window)

            请务必下载"Stable Releases"中的"Windows Installer(自己电脑的运行位数)"

            说明 : Windows embeddable package (64-bit), 下载下来的是.zip格式的安装程序
            Windows installer (64-bit), 下载下来的是.exe格式的安装程序, 我个人认为后者比较方便

            # 二、打开exe文件, 开始自定义安装

            1. 打开双击python-3.9.2-amd64.exe文件(因为版本不同, 所以下载文件名也不同)

            2. 勾选Add Python 3.9 to PATH，然后再点击Customize installation进入到下一步； (opens new window)

            3. 进入Optional Features后，不用操作，请务必全选，直接点击Next； (opens new window)

            4. 点击Browse进行自定义安装路径，自定义安装主要是为了节省C盘空间，点击install安装； 请注意! python需要被安装在全英文路径中, 不然后续会很麻烦!!!
              (opens new window)

            5. 点击Disable path length limit（修改电脑中对程序字符长度的限制）； (opens new window)

            6. 点击"close"即可完成

            # 三、确定Python是否安装完成

            • 在"开始"菜单中, 打开python IDLE, 即可打开python自带的编辑器
            • 按win+R, 输入框中输入cmd, 输入python, 即可在命令行工具中使用python

              恭喜你! 完美的安装好了python环境~

            # 如何安装第三方库

            • 按win+R, 输入框中输入cmd, 输入
            pip install 想安装的库名称
例:  pip install requests
            • 查看已安装的第三方库
            pip list
            • 用镜像下载第三方库(强烈推荐)
            # 由于python的站点在国外, 下载一些大型第三方库往往困难重重, 
# 但是我们可以用清华大学提供的镜像链接进行下载! 例如:
pip install -i https://pypi.tuna.tsinghua.edu.cn/simple 你需要下载的库名
