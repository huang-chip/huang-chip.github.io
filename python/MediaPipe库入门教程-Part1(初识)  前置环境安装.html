        # MediaPipe库入门教程-Part1(初识) : 前置环境安装

        # MediaPipe官方的自我介绍

        • MediaPipe 是一款由Google开发并开源的数据流处理机器学习应用开发框架。

        • MediaPipe支持跨平台开发, 目前支持的语言有:Android, ios, C++, Python, JavaScript

        • MediaPipe内置的功能包括但不限于 : 人脸检测, 面网格, 手部监测, 全身姿态检测, 全身整体检测(面网格+姿态检测+手部), 背景删除等

        • MediaPipe有以下四个优点 :

          1. 内置快速 机器学习 (ML) 推理 和 APU加速处理器(完美融合CPU和GPU) 即使在普通硬件上也是如此

          2. MediaPipe给出了统一解决方案, 适用于 Android、iOS、桌面/云、Web 和 物联网

          3. 即用型解决方案：使用了姿态识别的尖端技术和机器学习解决方案, 可以发挥出框架(framework)的满功率运转

          4. 免费和开源： Apache 2.0 下的框架和解决方案，完全可扩展和可定制

        编者将用python语言为基础, 为大家呈现MediaPipe主要功能的实现。

        # 运行代码前的准备工作

        请确保你已经安装了Python, 且Python的安装环境是纯英文路径, 会使用pip安装第三方库

        如果你没有安装Python, 可以参照此教程 : Python环境三分钟安装指南 来安装Python环境

        作者还提供了编程文档(jupyter文件), Jupyter超级好用, 你一定要尝试一下,可以参照此教程 : Python环境三分钟安装指南 来安装Jupyter

        第三方库的安装:
        按win+R, 输入框中输入cmd, 点击确定, 在命令行工具中输入 :

        pip install -i https://pypi.tuna.tsinghua.edu.cn/simple opencv-python
pip install -i https://pypi.tuna.tsinghua.edu.cn/simple mediapipe
        恭喜你完成了学习MediaPipe的第一步:认识它, 配置它的生长环境
