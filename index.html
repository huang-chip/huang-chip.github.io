    1. 使用黑名单而不是白名单或全局代理 : 使用GFW黑名单代理模式的优点之一，是节省购买的V2Ray节点流量，因为只有少量被墙封锁的网站走代理，其他的全部直连。

    • 基础知识

    • C#的多行注释是/**/ 单行注释是//
    • (必不可少)在任何 C# 程序中的第一条语句都是：using System;
    • 标识符必须以字母、下划线或 @开头,且大小写敏感,注意: @if 是有效的标识符，但 if 不是，因为 if 是关键字

    • C#的数据类型(三种:)

      • 值类型（Value types）
        就是我们常见的bool, int , float , char.都可以归类为值类型
      • 引用类型（Reference types）

        对象: (对象是值类型的终极基础类,因此可以被赋予任何类型的数据)

        动态（Dynamic）类型 /完全没看懂啊,过/
        字符串（String）类型
        (允许给变量分配任何字符串值),有两种赋值方式

        指针类型（Pointer types）
        (指针类型变量存储另一种类型的内存地址,用法同C和C++)

    • 类型转换(隐式和显氏)
      显氏是将其他类型变成基类(Object类型),不会数据丢失
      隐式就是用函数强制转换格式,比如

    • 给变量赋值(和C语言完全一致)
    • 给常量赋值(注意,常量const是不可修改的)

    • 运算符
      基础运算和其他语言一样(‘%’是取整除后的余数)
      自增自减运算 , 目前看到最好的解释

      c = a++:_____先将 a 赋值给 c，再对 a 进行自增运算。
      c = ++a:_____先将 a 进行自增运算，再将 a 赋值给 c 。
      c = a–:_____先将 a 赋值给 c，再对 a 进行自减运算。
      c = –a:_____先将 a 进行自减运算，再将 a 赋值给 c 。

    • 控制语句

    • 判断语句
      判断语句分为switch,if else嵌套,和三相判断

    • 循环语句
      循环语句分为while,for

    • 有关class的操作解释

    • C#封装
      所谓封装,就是在class里面声明变量的时候,前面加上一些修饰符(效果就像python里面的global a=5全局变量一样)//以下为解释
      (1) Pubilc ：任何公有成员可以被外部的类访问。
      (2) Private ：只有同一个类中的函数可以访问它的私有成员。
      (3) Protected ：该类内部和继承类中可以访问。
    • C# 方法
      所谓方法,就是用class里面的语句封装成一个函数
